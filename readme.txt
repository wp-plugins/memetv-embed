memeTV-embed allows you to embed videos created with memeTV.

== Description ==

memeTV-embed allows you to insert memeTV videos in your posts.
Plugin adds additional button to your blog's posts editor.
The only thing you need to do is grab videos URL, set up size and you're done!

== Installation ==

Upload the memeTV-Embed plugin to your blog, Activate it, start using!

== Screenshots ==

1. Go to Plugins > Add New > Upload. Select the ZIP file and hit "Install Now" button.
2. After succesfull installation, activate the plugin.
3. Use ConnecTV Embed button during Post creation.
4. Enter Video Page URL, select size and hit "Insert Code" button.
5. Voila! That's it!
